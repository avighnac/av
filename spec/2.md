## Goal

The goal is to get my compiler to successfully compile this, as for now at least

```av
int32 STDOUT_FILENO = 1;

void* alloc(int32 size);
void dealloc(void* mem);

int32 read(int32 fileno, void* mem, int32 bytes);
int32 write(int32 fileno, void* mem, int32 bytes);

int32 main();

int32
main()
{
  int8 c;
  c = 65_8;
  void* ptr;
  ptr = &c;
  int32 succ;
  succ = write(STDOUT_FILENO, ptr, 1_32);
  return 0_32;
}
```

Notice, for example, that

```av
int32
main()
{
```

is spread out across lines.

Anyway, so,

1. I tried doing something similar to what I did for arithmetica's algebra parser, that is, split on the 'top level root character' (for example in `a+b*c`) it's the +, that is, the one that happens *last*. 
2. That probably wouldn't have failed, but it was hard.

So I think a better approach would be to write a tokenizer first. 

I got that working today; so it tokenizes the above program to:

