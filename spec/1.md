## Primitive types

- `int8` (8-bit integer)
- `int32` (32-bit integer)
- `int64` (64-bit integer)
- `void*` (can be cast to anything)
- `int8/32/64*`

## Operators

- `*[identifier]` is a dereference
- `&[identifier]` gets the memory address

## Referencing/dereferencing

```
void* addr;
addr = 10;
int8 x;
int8* tmp = addr;
x = *tmp;
void* addr2;
addr2 = &x;
```

## Preprocessor

- `include "path"` copy-pastes the contents of the file in path after resolving dependencies

## Primitive variables (in stdlib.avh)

int32 STDIN_FILENO = 0;
int32 STDOUT_FILENO = 1;

## Primitive functions (declared in stdlib.avh)

```
void* alloc(int32 size);
void dealloc(void *mem);

int32 read(int32 fileno, void *mem, int32 bytes);
int32 write(int32 fileno, void *mem, int32 bytes);
```

## Entry point

```
int32 main() {
  
}
```

## Return

- `return [errorcode]`

## Integer literals

- `int32` => 123_32
- `int64` => 1982731_64
- etc...

## Target `.av` file to compile

```
void* alloc(int32 size);
void dealloc(void *mem);

int32 read(int32 fileno, void *mem, int32 bytes);
int32 write(int32 fileno, void *mem, int32 bytes);

int32_t main();

int32 main() {
  int8 c;
  c = 65_8;
  void* ptr;
  ptr = &c;
  int32 succ;
  succ = write(STDOUT_FILENO, ptr, 1);
  return 0_32;
}
```
